# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Требования

- python3
- Файл `wines.xlsx` cо списком вин и прочих напитков
- Файл `template.html` с кодом для генерации страницы сайта

## Запуск

- Скачайте код
- Установите зависимости 
```
pip install -r requirements.txt
```
- Запустите сайт командой 
```
python3 main.py
```
если у вас есть другой файл на замену `wines.xlsx` добавьте ключ `--path`
```
python3 main.py --path путь/ваш_файл.xlsx
```
- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
